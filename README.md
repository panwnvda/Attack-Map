# ATT&CKMAP

ATT&CKMAP is a cybersecurity knowledge base that organizes **methodologies, playbooks, and offensive security cheat sheets** according to the **MITRE ATT&CK framework**.

The goal of this project is to provide a **structured reference for red team operators and penetration testers** by mapping practical techniques to the real-world adversary behaviors defined by the **MITRE ATT&CK matrix**.

---

## Overview

ATT&CKMAP provides a curated collection of:

* Offensive security **methodologies**
* **Red team playbooks**
* **Cheat sheets and commands**
* **Technique-specific notes**
* **Operational workflows**

All content is organized by **ATT&CK tactic and technique** to make it easier to understand **how individual techniques fit into a complete attack chain**.

---

## Purpose

The objective of ATT&CKMAP is to bridge the gap between:

* **Theoretical frameworks**
* **Real-world offensive security operations**

By mapping practical knowledge to the ATT&CK framework, this project helps operators:

* Understand **how techniques relate to adversary behavior**
* Build **structured attack paths**
* Improve **red team planning and execution**
* Quickly reference **commands, tools, and workflows**

---

## Framework

ATT&CKMAP is structured around the **MITRE ATT&CK framework**, which categorizes adversary behavior into tactical stages of an intrusion.

Core tactics include:

* Reconnaissance
* Resource Development
* Initial Access
* Execution
* Persistence
* Privilege Escalation
* Defense Evasion
* Credential Access
* Discovery
* Lateral Movement
* Collection
* Command and Control
* Exfiltration
* Impact

Each tactic contains mapped techniques with supporting notes, methodologies, and references.
